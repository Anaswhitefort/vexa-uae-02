# What is this?

hpp(<a href="https://twitter.com/osaremochi" style={{color: "#fff"}}>@osaremochi</a>)のポートフォリオサイトです。

テンプレートなどは使わずに全て自作しました(デザインと背景も！)(右上の 🍔 メニューも自作です、頑張りました)。

以下使用技術を思いつくだけ書いてみます。

<br />

-   React.js
-   Gatsby.js (Blazing fast🔥 static site generator)
-   GraphQL
-   MDX (JSX in Markdown)
-   PWA (Progressive web application)
-   Netlify (static site hosting & CI/CD)
-   Netlify CMS
-   WebGL
-   Three.js (react-three-fiber)
-   react-spring (react's animation library)

<br />

PWA なので、PC からアクセスされた方はブラウザのアドレスバーの右のプラスマークを、iPhone の方は safari なら下のバーの真ん中のアイコンをタップして、その中のホーム画面に追加を押すことでローカルに PWA としてサイトを落とせます (iPhone ユーザーなので android はわかりません)。

適度に更新していく予定なので興味があればぜひ！
